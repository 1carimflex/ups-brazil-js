ups-brazil-js

UPS Brazil Javascript API Client

A JavaScript library to interface UPS Brazil API for calculate quote, it works in the browser and with Node.js. The documentation can be found here.

  originZipCode,
  destinationZipCode,
  packageData,
  invoiceValue,
  timeout
